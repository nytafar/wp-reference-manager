A modern block editor plugin for managing references, citations, and footnotes with support for PMID, DOI, URL, and ISBN.

== Description ==

WP Reference Manager is a powerful yet easy-to-use reference management plugin for WordPress. It integrates seamlessly with the block editor (Gutenberg) to help you manage citations and references in your academic or professional content.

Features:
* Add citations using PMID, DOI, URL, or ISBN
* Automatically fetch metadata from various sources
* Insert citations inline with your text
* Generate formatted bibliographies
* Support for multiple citation styles
* Clean and modern UI integrated with the block editor

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wp-reference-manager` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->WP Reference Manager screen to configure the plugin

== Frequently Asked Questions ==

= Which citation formats are supported? =

The plugin supports various citation formats including APA, MLA, Chicago, and more.

= Can I import references from other reference managers? =

Currently, the plugin supports adding references manually or via identifiers (DOI, PMID, ISBN, URL). Import functionality from other reference managers is planned for future releases.
